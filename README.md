# searls-game-of-life

h/t [Justin Searls](https://twitter.com/searls)

## getting started

Clone the repo:

```
$ git clone git@github.com:wpannell/searls-game-of-life.git
```

## requirements

* JDK 8
* Maven 3

## install

```
$ cd searls-game-of-life
$ mvn install
```

## intellij

* import as maven project
